## Репозиторий автотестов для страницы генератора виджета 3snet

Этот репозиторий содержит автоматизированные тесты для страницы [https://dev.3snet.info/eventswidget/](https://dev.3snet.info/eventswidget/), реализованные с использованием **Playwright** и **TypeScript**.

Охват тестов:

Тесты проверяют базовую работоспособность (Smoke Testing) страницы виджета:

1.  Доступность ресурса: Проверка HTTP-статуса ответа (200 OK).
2.  Целостность DOM: Проверка наличия заголовка страницы (Title).
3.  Отображение контента: Верификация того, что список событий или основной контейнер виджета корректно загружается и виден пользователю.
4.  Отсутствие JS-ошибок: В консоль выводятся ошибки браузера, если они возникают во время загрузки.

Примечание: В коде использованы универсальные селекторы для поиска карточек событий. Если верстка на dev-стенде изменится, рекомендуется обновить локаторы в `src/pages/EventsWidgetPage.ts`.

Запуск проекта:

Для запуска тестов необходимо наличие [Node.js](https://nodejs.org/) (версии 16 или выше).

Для запуска автотестов выполните команду в корне проекта:
npm test

Для просмотра отчета выполните команду:
npx playwright show-report

Установка зависимостей:

Выполните команду в корне проекта:

npm install && npx playwright install --with-deps
